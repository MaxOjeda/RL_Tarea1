Para replicar los experimentos de esta tarea se decidió separarla en 3 archivos del tipo "main" ejecutables, uno para cada agente: incrementalAgent, OptimisticAgent y GradientAgent.
Al ejecutar estos códigos se ejecuta cada agente y se muestran las gráficas correspondientes a cada experimento según lo solicitado. Estos son los comandos para ejecutar:

py IncrementalMain.py
py OptimistMain.py
py GradientMain.py

Nota:   1. La ejecución tarda alrededor de 3 minutos por experimento 
        2. En estos comando el "py" puede variar dependiendo de la versión de python o el entorno local por "python" o "python3".
